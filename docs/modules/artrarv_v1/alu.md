# Arithmetic Logic Unit (ALU)

## Status

✅ Completed

## Description

The ALU performs arithmetic, logical, comparison, and shift operations required by the RV32I ISA.

## Supported Operations

- ADD
- SUB
- AND
- OR
- XOR
- SLL
- SRL
- SRA
- SLT
- SLTU

## Features

- Parameterized 32-bit datapath
- Combinational implementation
- Zero flag generation
